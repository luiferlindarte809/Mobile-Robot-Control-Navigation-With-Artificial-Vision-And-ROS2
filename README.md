# Mobile-Robot-Control-Navigation-With-Artificial-Vision-And-ROS2
Implementación de un sistema de navegación para un robot móvil autónomo. Incluye arquitectura en ROS2, percepción mediante visión artificial (YOLO) y sensórica, y diseño de estrategias de control óptimo (LQR/LQG) y no lineal con SLAM. Proyecto de grado.
